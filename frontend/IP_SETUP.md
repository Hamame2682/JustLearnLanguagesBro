# iPhoneからアクセスする場合の設定

## 方法1: 環境変数で設定（推奨）

`frontend/.env.local` ファイルを作成し、PCのIPアドレスを設定：

```bash
NEXT_PUBLIC_API_URL=http://192.168.1.45:8000
```

**PCのIPアドレスの確認:**
- Windows: `ipconfig` コマンドを実行
- Mac/Linux: `ifconfig` または `ip addr` コマンドを実行

環境変数を設定したら、フロントエンドサーバーを再起動してください。

## 方法2: 自動検出（デフォルト）

環境変数を設定しない場合、アプリは自動的に現在のホスト名からAPI URLを構築します。

- iPhoneから `http://192.168.1.45:3000` でアクセス
- → APIは自動的に `http://192.168.1.45:8000` に設定される

この方法なら、IPアドレスが変わっても自動的に対応できます。

## トラブルシューティング

### 「Load Failed」エラーが出る場合

1. **PCのIPアドレスが正しいか確認**
   ```bash
   # Windows
   ipconfig
   
   # Mac/Linux
   ifconfig
   ```

2. **バックエンドが起動しているか確認**
   ```bash
   cd backend
   uvicorn main:app --reload --host 0.0.0.0 --port 8000
   ```
   `--host 0.0.0.0` が重要です。これがないと外部からアクセスできません。

3. **ファイアウォールの設定を確認**
   - Windows: コントロールパネル → ファイアウォール → ポート8000を許可
   - Mac: システム環境設定 → セキュリティとプライバシー → ファイアウォール

4. **同じWi-Fiネットワークに接続されているか確認**
   - PCとiPhoneが同じWi-Fiネットワークに接続されている必要があります
