# IPアドレスをハードコードする方法

もし環境変数を使わずに、直接IPアドレスをハードコードしたい場合：

## 手順

1. `frontend/lib/api.ts` を開く

2. `getApiUrl()` 関数を以下のように変更：

```typescript
export function getApiUrl(): string {
  // クライアントサイド（ブラウザ）の場合
  if (typeof window !== 'undefined') {
    // ハードコード: PCのIPアドレスを直接指定
    return 'http://192.168.1.45:8000';  // ← ここをPCのIPアドレスに変更
  }
  
  // サーバーサイドの場合
  return 'http://192.168.1.45:8000';  // ← ここも同じIPアドレスに変更
}
```

**注意:** 
- `192.168.1.45` を実際のPCのIPアドレスに置き換えてください
- IPアドレスが変わった場合は、このファイルを再度編集する必要があります
- 環境変数を使う方法（`.env.local`）の方が柔軟性が高いので推奨します
